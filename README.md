# HopeHarbor Foundation Website

A modern and elegant NGO website designed for **HopeHarbor Foundation**, focused on presenting social initiatives, impact stories, donation opportunities, and community programs through a highly interactive and responsive interface.

---

## 🌟 Features

- Responsive design for desktop and mobile devices
- Smooth scroll animations and reveal effects
- Interactive custom cursor
- Animated statistics and timeline sections
- Hero section with floating background effects
- Programs and impact showcase
- Team section
- Donation form interface
- Contact form
- Modern footer with social links
- Pure HTML, CSS, and JavaScript implementation

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
